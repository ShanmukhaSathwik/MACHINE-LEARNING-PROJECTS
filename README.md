# 🤖 Machine Learning Projects
This repository contains machine learning projects built using Python, NLP, and AI models.

---

## 📁 Projects

| Project | Description | Tech Used |
|---|---|---|
| [Clinical Trial Protocol Extractor](./Clinical-Trial-Protocol-Extractor/) | Extracts & summarizes key sections from COVID-19 clinical trial PDFs using NLP and BART | Python, HuggingFace, BART, DistilBERT |
| [Emotion Recognition](./Emotion_Recognition/) | Multimodal emotion recognition on the MELD dataset using DistilBERT + attention fusion. Outperforms baseline on 5/7 emotion classes with weighted F1 of 0.5523 | Python, TensorFlow, DistilBERT, LSTM, librosa |
| [PocketDoc](./Pocketdoc/) | AI-powered medical triage assistant that classifies symptom severity, asks adaptive follow-up questions, and generates structured clinical reports using RAG | Python, Groq, LLaMA 3.3, FAISS, SentenceTransformers |
